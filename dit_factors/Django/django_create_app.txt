* We created an App. In order to render the app using urls we need to includethe app in our main project so
that urls redirected to that app can be rendered

We go to Dit_Timetable->cbt_viewer->settings.py
and we change the following sector 

INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles'
]

in to this
INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
